# Quality Assurance and Calibration

Static GitHub Pages app for `Unit2_Section3_QA`.

Built using `Lecture_App_Master_Prompt_and_Workflow.docx` as the governing workflow.

## Includes

- Lecture-sequence tabs based on slide ranges.
- Topic-specific interactive lab: Calibration Curve Builder, Control Chart Lab, Spike Recovery and LOD/LOQ Coach.
- Slide Studio with 47 slide images and accessible transcripts.
- Original lecture PDF download.
- Practice cards extracted from instructor-provided practice materials: 55 cards.
- No Word files exposed in the student-facing app.
- Keyboard navigation, visible focus, live feedback, text-first mode, graph summaries, and data tables.
